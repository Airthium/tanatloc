# Tanatloc

Using server-side rendering (SSR).

## Dev

Browser:

```
yarn install
yarn dev
```

Electron:

```
yarn install
yarn electron-dev
```

## Build

Browser:

```
yarn install
yarn build
yarn start
```

Browser (static build):

```
yarn install
yarn export
```

Electron

```
yarn install
yarn electron-build
```
